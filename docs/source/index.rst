Welcome to pscapi's documentation!
==================================

| `pscapi <https://github.com/satamame/pscapi>`_ は、`playscript <https://github.com/satamame/playscript>`_ という Python パッケージの機能を Web API にするための `FastAPI <https://fastapi.tiangolo.com/ja/>`_ アプリケーションです。
| これにより、Python を使わずに台本のフォーマットを変換したり、Web 上にある台本を共有したりさせる事ができます (pscapi 自体は Python を使います)。
